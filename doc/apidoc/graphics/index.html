tewi doc - Graphics module

Modules

[RS] = Requires API tag specialization. This means that what you see is just an interface, the implementation is different for each API tag. The lack of an implementation will result in a compiler error.

[SS] = Supports API tag specialization. This means that the class has a general implementation that works with any API tag, but it's still possible to override said implementation by specializing it with an user-provided tag. This is officially supported only for user-defined tags. Specializing with any tag in tewi::API::* is unsupported and discouraged as it may cause undefined behaviour.
